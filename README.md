# Visrodeck-Tasks
VISRODECK Tasks is a modern desktop task management software built with Python and CustomTkinter. It provides a clean dashboard interface to help users organize, track, and manage daily tasks efficiently.   Well This Is The Main Idea, As I Learn More Explore More I will Keep Improving this!

Features

Add and delete tasks

Mark tasks as completed

Set due dates and priorities

Dashboard stats (Total / Completed / Pending)

Dark modern UI

Local file storage (JSON)

Tech Used

Python

CustomTkinter

JSON storage

Run with Python
pip install customtkinter
python VisrodeckTasks.py

Build EXE (CMD)
> pip install pyinstaller
> pyinstaller --onefile --windowed --collect-all customtkinter VisrodeckTasks.py
